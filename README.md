# Gerenciador de Tarefas

- Sistema de Gerenciador de Tarefas utilizando Next.js e Firebase
- Opção de Comentar em Tarefas Públicas

## Tecnologias

⚛️ Next.js 13

📂 Google Firebase

🔒 Next-Auth

🎨 TailwindCss
